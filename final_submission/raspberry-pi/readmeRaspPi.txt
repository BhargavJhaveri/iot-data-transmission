Code to Raspberry Pi:

Real time data transfer - The code needed to run is realTimeRaspPi.py
You will need Python 2.7 or higher to execute the code. 
The following commands need to be executed to run the program.
1) python realTimeRaspPi.py

File transfer application -  The code needed to run is fileTransmissionRaspPi.py
You will need Python 2.7 or higher to execute the code.
The following commands need to be executed to run the program.
1) python file_transfer_rasp_pi.py
